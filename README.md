# Ross Maritime Supply Limited

Ross Maritime Supply Limited is an online e-commerce platform designed to facilitate the purchase and supply of maritime equipment and goods. This project is a web application built with vanilla HTML, CSS, and JavaScript, offering a user-friendly interface that allows customers to browse products, manage their cart, and process orders.

    ```bash
    git clone [https://github.com/josephnjogu535-ctrl/Ross-Maritime-Supply-Limited.git](https://github.com/josephnjogu535-ctrl/Ross-Maritime-Supply-Limited.git)
    ```

    ```bash
    cd Ross-Maritime-Supply-Limited
    ```

3.  Open the application:
    * Locate the `index.html` file in your file explorer and double-click it to open it in your default web browser.
    * Alternatively, if you have VS Code with the Live Server extension, you can right-click `index.html` and select "Open with Live Server".
